## 📖 About

CyberNexus is a **free, open-source cybersecurity education platform** built as a single, self-contained HTML file — no backend, no login, no paywall. It covers 25 structured modules from core fundamentals to advanced topics, complete with real-world examples, diagrams, interview prep, and curated YouTube resources.

Designed for:
- 🎓 Students entering the cybersecurity field
- 💼 Professionals prepping for certifications (CompTIA, CEH, CISSP)
- 🔁 Career-switchers wanting a structured learning path
- 🧠 Anyone who wants to understand how digital security actually works

## 🚀 Getting Started

### Option 1 — Use the Live Site
Visit **[https://sid-3110.github.io/Cyber-Nexus-CSF-platform/](https://sid-3110.github.io/Cyber-Nexus-CSF-platform/)** directly in any browser.

### Option 2 — Run Locally
```bash
# Clone the repo
git clone https://github.com/sid-3110/Cyber-Nexus-CSF-platform.git

# Open in browser (no server needed)
open index.html
# or on Windows:
start index.html
```

That's it. No `npm install`. No build process. Just open and learn.

## 🤝 Contributing

Contributions are welcome! Here's how:

1. **Fork** the repository
2. **Create a branch**: `git checkout -b feature/add-module-26`
3. **Make your changes** in `index.html`
4. **Commit**: `git commit -m "feat: add Module 26 - Cloud Native Security"`
5. **Push**: `git push origin feature/add-module-26`
6. **Open a Pull Request**

### Contribution Ideas
- Add new modules (Cloud Native, DevSecOps, AI Security)
- Expand interview prep questions
- Add more YouTube resource links
- Translate to other languages
- Improve accessibility (ARIA labels, keyboard nav)
- Add dark/light mode toggle
