# ⬡ CyberNexus — Cybersecurity Fundamentals

![Open Source](https://img.shields.io/badge/Open%20Source-✓-blue) ![Modules](https://img.shields.io/badge/Modules-25-blue) ![Free](https://img.shields.io/badge/Free-Forever-brightgreen) ![Beginner Friendly](https://img.shields.io/badge/Beginner-Friendly-blue)

A comprehensive, open-source cybersecurity fundamentals course designed for **absolute beginners**. Covers everything from core security principles to advanced threat intelligence — all in a single interactive HTML file. No install, no login, no cost.

---

## 🚀 Features

- 25 structured modules from beginner → industry-ready
- 200+ definitions for interview preparation
- Interactive diagrams and visual explanations
- YouTube video resources for every module
- Interview prep questions per topic
- Searchable quick-reference glossary
- 100% offline capable — single HTML file
- Sleek black & blue professional theme

## 🛠️ How to Use

1. Clone or download the repo
2. Open `cybernexus.html` in any browser
3. No server, no dependencies, no install needed
```bash
git clone https://github.com/sid--3110/cybernexus.git
cd cybernexus
open cybernexus.html
```
